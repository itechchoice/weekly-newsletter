# Die Yi 技术周刊 - AI Agent 指南

本文档为 AI Agent 提供周刊创作指南，确保内容风格一致。

## 项目概述

Die Yi 技术周刊是公司内部的技术分享周刊，内容包括：

- 有趣的 GitHub 开源项目
- Reddit/技术社区热门话题
- 实用的 AI 工具和生态
- 编程最佳实践和规范
- 设计最佳实践和规范

## 文件结构

```

├── YYYY-MM-DD-weekly-NNN.md  # 周刊文章

```


## 写作风格

1. **标题要吸引眼球**：副标题用 `>` 引用块，要有冲击力
2. **痛点先行**：先描述问题，再给解决方案
3. **口语化表达**：用"你"，不用"您"；用问句引起共鸣
4. **加入内部心得**：用 `_斜体_` 加俏皮话，如"🤫 悄悄告诉你：我们已经在用了"
5. **表格展示特性**：功能列表用表格，更清晰
6. **代码要可复制**：提供完整可运行的示例
